## PR checklist

- [ ] One line, neutral tone, link to repo (and npm if relevant)
- [ ] Placed in the correct alphabetical order within the section
- [ ] No marketing superlatives ("best", "fastest")
- [ ] Mention what makes it different in ≤15 words

## After a list merges
